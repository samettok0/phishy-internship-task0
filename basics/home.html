Welcome to the Phishy Internship Task 0

This is a simple Node.js/Express.js and JavaScript demonstration project.

Features:
  • Prints messages and environment info to the console
  • Demonstrates synchronous and asynchronous file reading
  • Uses Node.js EventEmitter for custom events
  • Serves this HTML page using Express.js
  • Imports and displays a custom module

How to Run

  1. Install dependencies: npm install
  2. Start the server: node index.js
  3. Open http://localhost:3000 in your browser

Project Files

  • index.js – Main server and logic
  • my-module.js – Custom module
  • hello.txt – Text file for file reading demo
  • home.html – This HTML page
